Bootstrap components in Vue.js

Typeahead

<typeahead id="typeahead-input" placeholder="Type a name" :items="typeaheadItems"></typeahead>

Props:

NameTypeDescription
valueStringInitial value of the input field.
itemsArrayAn array of items that will be used to display autocomplete suggestions.
idStringThe id attribute of the input element.
placeholderStringThe placeholder attribute of the input element.

Custom templates

You can use inline templates to customise the default template of any of the components.

<component inline-template>
  <div>...your custom template goes here...</div>
</component>
